> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace 9 sites that compute `sum/n` arithmetic means of berry / wine chemistry with `weightedMean(rows, valueKey, '_weight')`, weighted by each lot's `tons_received` from `mediciones_tecnicas`. Lots without a matching medicion fall back to weight 1. The displayed numbers will shift toward the chemistry of larger commercial lots (which is what users actually care about).

**Architecture:** New pure module `js/aggregations.js` exports `weightedMean` and `peakBy`. `js/dataLoader.js` builds a `Map<lotCode, tons>` from `medicionesData` and tags every berry + wine sample with `_weight` after enrichment. Skipped in demo mode (synthetic data, irrelevant weighting). 9 callsites replace their inline `avg = arr => arr.reduce(...) / arr.length` with `weightedMean(rows, key)`.

**Tech Stack:** Vanilla JS ES modules, node:test for unit tests.

```javascript
// js/aggregations.js
// Pure aggregation utilities. No DOM, no globals.
// Used by KPIs, charts, maps for tonnage-weighted means via
// mediciones_tecnicas.tons_received (tagged onto sample rows as _weight
// in dataLoader._enrichData; see Wave 1 #1).

/**
 * Weighted arithmetic mean.
 * @param {Array<object>} rows
 * @param {string} valueKey — property name to average (e.g. 'brix')
 * @param {string} [weightKey='_weight'] — property name for the weight
 * @param {object} [opts]
 * @param {number} [opts.fallbackWeight=1] — used when row[weightKey] is null/0/NaN
 * @returns {number|null} weighted mean, or null if no valid rows
 */
export function weightedMean(rows, valueKey, weightKey = '_weight', { fallbackWeight = 1 } = {}) {
  let num = 0;
  let den = 0;
  for (const r of rows) {
    const v = r[valueKey];
    if (v === null || v === undefined || Number.isNaN(v) || typeof v !== 'number') continue;
    const rawW = r[weightKey];
    const w = (typeof rawW === 'number' && rawW > 0 && !Number.isNaN(rawW)) ? rawW : fallbackWeight;
    num += v * w;
    den += w;
  }
  return den > 0 ? num / den : null;
}

/**
 * Returns the row with maximum value at `key`. Ties keep first encountered.
 * Skips null / undefined / NaN values. Returns null on empty/all-skipped.
 */
export function peakBy(rows, key) {
  let best = null;
  let bestVal = -Infinity;
  for (const r of rows) {
    const v = r[key];
    if (v === null || v === undefined || Number.isNaN(v) || typeof v !== 'number') continue;
    if (v > bestVal) { bestVal = v; best = r; }
  }
  return best;
}
```
