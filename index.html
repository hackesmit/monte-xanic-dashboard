Guía de Uso — Dashboard Vendimia

Cómo Iniciar

El dashboard carga datos automáticamente desde archivos JSON pre-extraídos en la carpeta data/. Si los archivos no están presentes, se mostrará la pantalla de carga para subir datos manualmente.

Para servir el dashboard localmente, ejecute:

python -m http.server 8080

Luego abra http://localhost:8080 en su navegador.

Cargar / Actualizar Datos

Haga clic en "Recargar Datos" en la barra lateral para abrir la pantalla de carga. Puede arrastrar y soltar archivos o usar el botón de selección.

Archivo principal:

  • Exportación WineXRay (.csv o .xlsx) — Un solo archivo con todos los datos. El dashboard separa automáticamente muestras de bayas (Sample Type = Berries) y muestras de vino (Aging Wine, Young Wine, Must, etc.).
  • Movimientos de tanque (opcional) — Archivo de contexto sobre mezclas y movimientos.

Formato de Datos Requerido

Exportación WineXRay (.csv o .xlsx):

Exportación directa de WineXRay. La primera fila debe ser encabezados. Valores nulos se representan con - o —. El archivo contiene tanto datos de bayas como de vino, diferenciados por la columna Sample Type. Columnas utilizadas:

Sample Id Sample Type Sample Date Vintage Variety Appellation Vessel Id DaysPostCrush Brix pH tANT fANT pTAN IPT Berry Fresh Weight

Vistas del Dashboard

  • Bayas — Gráficas de dispersión de Brix, pH, tANT, Acidez y Peso vs Días Post-Envero. Comparativo por varietal y distribución por origen.
  • Vino — Tablas de recepción de tanque y prefermentativos con KPIs. Filtre por varietal, proveedor/origen y tipo de uva.
  • Extracción — Comparación baya→vino de antocianinas totales. Muestra porcentaje de extracción fenólica.
  • Vendimias — Gráficas de comparación 2024 vs 2025 y tablas de resumen con promedios por parámetro y varietal.

Controles y Filtros

  • Tipo de Uva — Filtre entre tintas, blancas o todas.
  • Varietal — Haga clic en chips para seleccionar/deseleccionar varietales. Los chips tienen borde de color indicando tinta (rojo) o blanca (amarillo).
  • Origen — Filtre por viñedo/origen.
  • Lote — Busque y filtre por lote específico.
  • Color por — Cambie el color de las gráficas entre "Por Varietal" y "Por Origen".
  • Conectar Líneas — Active/desactive líneas entre puntos de datos.
  • Leyenda — Haga clic en items de la leyenda para ocultar/mostrar series.
  • Modo Claro/Oscuro — Use el botón de luna/sol en la esquina superior derecha.

Regenerar Datos JSON

Para actualizar los datos pre-extraídos, coloque los archivos Excel en la carpeta del proyecto y ejecute:

python extract_data.py

Esto generará los archivos en data/ y el dashboard los cargará automáticamente en la siguiente visita.
